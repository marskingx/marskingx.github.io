## 🎯 系統特色

### ⭐ 5 碼版本號系統
- **格式**: `major.minor.patch.content.log`
- **第 5 碼**: 專門追蹤 AI 協作日誌版本

### 🚀 智能發布功能
- 自動區分公有/私有檔案
- AI 記憶檔案自動路由到私有庫
- 支援中文暗號指令

### 🧠 AI 協作整合
- 自動記錄 AI 工作日誌
- 多 AI 協作透明化
- 完整的版本追蹤

## 🔗 快速開始

```bash
# 查看當前版本
npm run version:show

# 智能提交 (僅本地)
npm run 上版

# 智能發布 (含部署)
npm run 上版&佈署

# 記錄協作日誌
npm run 記憶
```
